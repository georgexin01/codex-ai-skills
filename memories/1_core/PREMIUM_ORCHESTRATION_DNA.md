# 🎼 PREMIUM ORCHESTRATION DNA (V2.0 - KINETIC EVOLUTION)

## 🎯 The Core Philosophy
Premium design is **alive**. Version 2.0 moves beyond static layouts into **Cinematic Storytelling**. We use "Liquid" interfaces and "Kinetic" typography to capture the user's attention instantly.

---

## 🏗️ 1. THE STITCH ENGINE (V2.0 - LIQUID & KINETIC)

### 🧊 1.1 Liquid Glass Aesthetic
The "Liquid Glass" look is achieved through high-vibrancy translucency.
- **Tokens**: 
  - `background: rgba(255, 255, 255, 0.03);`
  - `backdrop-filter: blur(40px) saturate(200%);`
  - `border: 1px solid rgba(255, 255, 255, 0.08);`
- **Effect**: Elements should feel like they are floating in a fluid, high-end environment.

### 📐 1.2 Tilt-Stage Orchestration (V2.1)
- **Concept**: A 3D-like depth created through deliberate rotation and scaling of core containers.
- **Rules**:
  - Root board MUST use `transform: rotate(-8deg) scale(0.78);`.
  - Cards inside the board should have a "Glassy" feel to contrast the dark background.
  - Media queries MUST adjust the scale (e.g., `.56` for mobile) to maintain readability.

### 🖋️ 1.2 Kinetic Typography (The "Viral" Rule)
- **Scale**: Headlines MUST be massive (`fs-100` to `fs-150`).
- **Length**: Keep it short. 1-3 words max for the Hero.
- **Font Pairing**: 
  - **The Hook**: A premium Serif (e.g., *Instrumental Serif*) for emotional weight.
  - **The Logic**: A clean Sans-Serif (e.g., *Inter*) for technical details.

## 🍌 2. THE NANO BANANA PIPELINE (V2.0 - MOTION FIRST)

### 🌊 2.1 Cinematic Motion Backgrounds
Since Gemini 3 Flash does not generate video directly, we **simulate** video loops:
1.  **The Expansion**: Ask Nano Banana 2 to expand images to `16:9` for edge-to-edge coverage.
2.  **The Simulation**: Use CSS `animation: ken-burns 20s infinite alternate;` on Nano Banana assets to create constant, slow movement.
3.  **The Video Wrapper**: If the user provides a direct `.mp4` link, it MUST be used as a 100% opacity background with no overlays to preserve visual purity.

## 🤝 3. THE V2.0 HANDSHAKE (The Workflow)

### 3.1 The "Viral" Generation Sequence
1.  **Mood Genesis**: Define the "Loop Theme" (e.g., Dark Tech, Minimalist Luxury).
