# 📏 [05] UI Standardization — The Compact Protocol

## 🎯 Objective
To create a high-density, professional mobile experience with "tight" layout ratios and clean white aesthetics.

## 📐 Geometry (Compact)
| Property | Value | Note |
|---|---|---|
| **Body Radius** | 20px | Refined from 40px for precise look. |
| **Card Radius** | 16px | Internal components. |
| **Header Height** | 64px | Sticky glass. |
| **Footer Height** | 96px | Floating glass. |
| **Safe Padding** | 20px (p-5) | Standard mobile rhythm. |

## 🎨 Theme (Light Premium)
- **Background**: `#F2F2F7` (System Gray 6)
- **Cards**: `#FFFFFF` (Pure White)
- **Primary**: `#1A1A1A` (Deepest Ink)
- **Accent**: `#007AFF` (iOS Blue)
- **Borders**: `#E5E5EA` (Thin hairline)

## 🕹️ Interaction
- **Stagger**: GSAP 0.05s stagger on card entries.
- **Feedback**: `active:scale-[0.98]` on cards.
- **Typography**: 10pt/12pt Body, 8pt Industrial Labels.
