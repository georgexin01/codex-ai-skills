# 🖼️ [06] View Scaffolding — The Sovereign App Protocol

## 🎯 Objective
To build the primary application screens with focus on **Visual Hierarchy** and **Context-Aware Layouts**.

## 🖥️ View Architectures
1.  **High-Density Dashboard (Admin)**:
    *   Top stats bar with large, bold numbers.
    *   Real-time fleet list with status indicators.
    *   Drawer-based task creation for non-intrusive flow.
2.  **Mission View (Rider)**:
    *   One-handed navigation (Actions at bottom).
    *   Large "Proof of Work" photo slots.
    *   Contextual state changes (Picking Up vs. Delivering).

## 🪄 Design Spells
- **Skeleton Loading**: Simulate data fetching with pulsing gray panels.
- **Micro-shadows**: Use different shadow depths to indicate stack hierarchy.
