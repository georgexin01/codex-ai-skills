# 🧬 [01] Handshake Genesis — The Sovereign Design Start

## 🎯 Objective
To establish a "Mental Lock" on the project's purpose before a single line of code is written. This step ensures that the AI and the User are perfectly synchronized on the **Visual Direction**, **Target Market**, and **Functional Goals**.

## 📖 The Protocol
1.  **Read the Master Blueprint**: You MUST read `BLUEPRINT.md` in the project root. Identify the "Project Type" (PHP, HTML, etc.) and the "Project DNA" (Colors, Fonts, Tone).
2.  **Define the Page's Soul**: 
    - What is the primary action we want the user to take? (Buy, Sell, Learn, Search).
    - What is the emotional response we are aiming for? (Trust, Excitement, Luxury, Speed).
3.  **DNA Synchronization**: 
    - Fetch the latest design tokens from `.codex/memories/1_core/UI_DNA_MASTER.md`.
    - Cross-reference with the project's specific palette (e.g., Carnews Orange #FF7101).
4.  **Reference Guard Handshake**:
    - Identify the `download-template/` version to refer to for original structure.
    - Declare that this reference will be UNTOUCHABLE and used only for logic extraction.
