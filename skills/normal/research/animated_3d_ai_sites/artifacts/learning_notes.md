# AI Learning Notes: Animated 3D AI Site Mastery

## Core Learnings
1.  AI-Generated Motion is the New Standard: High-end sites are moving away from static HERO images. Animated textures (Lava, Particles, 3D Fluid) generated by models like Nano Banana create a premium feel that justifies high-ticket prices ($2k–$5k).
2.  Reference-First Prompting: The fastest way to build a high-fidelity site is to "screenshot the competition" and feed it to a layout bot like Lovable.dev. This bypasses the blank-canvas phase.
3.  Agentic Refinement for the "Final 10%": Lovable/v0/Bolt are good for the 80% build. The final 20% (precision alignment, interactive hover states, specific hex-matching) is best handled by an agentic CLI like Claude Code.
4.  AntiGravity X Prompt Library: This is a specialized resource (`motionsites.ai`) that acts as a "Cheat Code" for React/Tailwind developers. It provides copy-pasteable blocks that are already polished and animated.
