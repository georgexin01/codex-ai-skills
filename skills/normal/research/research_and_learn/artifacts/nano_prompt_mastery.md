## 🧩 1. The Metadata Mapping Protocol
The core logic of "Mastery" prompting is treating an image as a structured object. By extracting the metadata of an image into a machine-readable JSON schema, you gain 100% surgical control.

### Metadata Extraction Prompt:
"Analyze this image and extract all technical and visual metadata into a structured JSON schema. Include objects, style, colors, lighting, composition, and optics (lens/angle)."

### Surgical Modification Workflow:
1. Metadata Map: Get the JSON baseline.
2. Key Target: Identify the specific field to change (e.g., `shirtcolor` or `lenstype`).
3. Control Edit: Manually update the JSON key or prompt the AI to modify that specific key while holding others constant.
4. Consistency Leak Protection: Download the new result and re-extract the Metadata Map to ensure the DNA hasn't drifted.

## 🖌️ 3. Visual Logic Tools
Gemini uses multimodal markers to refine localized areas.

- Brush Tool: Circle or draw arrows to highlight a specific object.
- Text Labels: Use text nearby the markers to specify the change (e.g., "turn this blue" or "add a watch here").
- Outpainting: Prompt "Generate a full-body image of this person [aspect ratio 9:16]" to expand original frames into cinematic body shots.
